## Installation

1. Grab the module.json url from the most recent release.
2. On your FoundryVTT instance click Add-On Modules from the front menu.
3. Click Install Modules
4. Paste the url in the Manifest URL box at the bottom and click install.
5. Enabled the module within your PF2e World
